# File Uploader API 调用说明

本文档详细说明如何调用 File Uploader 服务的各种 API 接口，包括签名生成、文件上传、删除等操作。

## 🔐 签名认证机制

File Uploader 使用 HMAC-SHA256 签名算法确保 API 调用的安全性。

### 签名生成规则

**重要**：所有客户端必须严格遵循以下规则：

1. **路径格式**：使用**未编码的原始路径**生成签名（包含中文等特殊字符）
2. **签名格式**：HMAC(32字节) + 随机数(8字节) = 40字节，转换为80个十六进制字符
3. **签名内容**：`路径 + 过期时间戳 + 随机数(十六进制)`
4. **密钥**：配置文件中的 `security.secret_key`

### 路径处理规范

- ✅ **正确**：`/api/v1/files/hxljzz/images/微信图片_example.jpg`
- ❌ **错误**：`/api/v1/files/hxljzz/images/%E5%BE%AE%E4%BF%A1%E5%9B%BE%E7%89%87_example.jpg`

**说明**：
- 客户端生成签名时使用未编码的原始路径
- HTTP请求时由HTTP库自动处理URL编码
- 服务端会自动解码URL后进行签名验证

### 签名生成示例

```go
package main

import (
    "crypto/hmac"
    "crypto/rand"
    "crypto/sha256"
    "encoding/hex"
    "fmt"
    "time"
)

func generateSignature(path, secretKey string, expiryDuration time.Duration) (string, int64, error) {
    // 计算过期时间戳
    expires := time.Now().Add(expiryDuration).Unix()
    expiresStr := fmt.Sprintf("%d", expires)

    // 生成8字节随机数
    nonce := make([]byte, 8)
    if _, err := rand.Read(nonce); err != nil {
        return "", 0, fmt.Errorf("生成随机数失败: %v", err)
    }

    // 构建签名内容：路径 + 过期时间 + 随机数
    signContent := path + expiresStr + hex.EncodeToString(nonce)

    // 计算HMAC
    h := hmac.New(sha256.New, []byte(secretKey))
    h.Write([]byte(signContent))
    hmacBytes := h.Sum(nil)

    // 组合最终签名：HMAC + 随机数
    finalSignature := append(hmacBytes, nonce...)

    // 转换为十六进制字符串
    signatureHex := hex.EncodeToString(finalSignature)

    return signatureHex, expires, nil
}

func main() {
    secretKey := "your-secret-key"
    path := "/api/v1/files/hxljzz/images/微信图片_example.jpg"  // 使用未编码的原始路径

    signature, expires, err := generateSignature(path, secretKey, time.Hour)
    if err != nil {
        fmt.Printf("Error: %v\n", err)
        return
    }

    fmt.Printf("Path: %s\n", path)
    fmt.Printf("Signature: %s\n", signature)
    fmt.Printf("Expires: %d\n", expires)
    fmt.Printf("URL: %s?expires=%d&signature=%s\n", path, expires, signature)
}
```

## 📤 文件上传 API

### 单文件上传

**接口地址**：`POST /api/v1/upload`

**请求参数**：
- `expires`：过期时间戳（查询参数）
- `signature`：签名值（查询参数）
- `file`：上传的文件（表单数据）
- `storage`：存储类型（表单数据，可选）
- `path`：自定义路径（表单数据，可选）

**支持的文件类型**：
- **图片格式**：JPG、PNG、GIF、WebP、**AVIF**、BMP、SVG、ICO
- **文档格式**：PDF、TXT、DOC、DOCX、XLS、XLSX、PPT、PPTX
- **压缩格式**：ZIP、RAR、7Z、TAR、GZ
- **音视频格式**：MP4、AVI、MOV、WMV、FLV、WebM、MKV、MP3、WAV、FLAC、AAC
- **通配符配置**：支持在配置文件中使用 `*`、`image/*`、`video/*` 等通配符模式

**请求示例**：
```bash
curl -X POST "http://localhost:8080/api/v1/upload?expires=1735977600&signature=abc123..." \
  -F "file=@example.jpg" \
  -F "storage=protected_images" \
  -F "path=/images/2024/example.jpg"
```

**响应示例**：
```json
{
  "success": true,
  "url": "http://localhost:8080/uploads/example.jpg",
  "thumbnail_url": "http://localhost:8080/uploads/example_Thumbnail.jpg",
  "filename": "example.jpg",
  "size": 102400,
  "content_type": "image/jpeg",
  "upload_time": 1704067200,
  "storage_type": "local"
}
```

**🎯 缩略图智能返回**：
- 满足生成条件：返回实际缩略图URL
- 不满足条件：返回原图URL作为thumbnail_url
- 功能禁用：返回原图URL作为thumbnail_url
- 简化调用端逻辑，无需额外判断

## 🗑️ 文件删除 API

### 单文件删除

**接口地址**：`DELETE /api/v1/files/{filepath}`

**说明**：`{filepath}` 参数支持包含完整路径的文件名，可以删除任意目录层级的文件。

**请求参数**：
- `expires`：过期时间戳（查询参数）
- `signature`：签名值（查询参数）
- `storage`：存储类型（查询参数，可选）

### 🔐 签名生成规则

**重要**：客户端生成签名时必须遵循以下规则：

1. **路径格式**：使用**未编码的原始路径**生成签名
2. **签名格式**：HMAC(32字节) + 随机数(8字节) = 40字节，转换为80个十六进制字符
3. **签名内容**：`路径 + 过期时间戳 + 随机数(十六进制)`

**正确示例**：
```javascript
// ✅ 正确：使用未编码的原始路径
const path = '/api/v1/files/hxljzz/images/20250710/微信图片_example.jpg';
const expires = Math.floor(Date.now() / 1000) + 3600;
const nonce = generateRandomBytes(8); // 8字节随机数
const signContent = path + expires + nonce.toString('hex');
const hmac = crypto.createHmac('sha256', secretKey).update(signContent).digest();
const signature = Buffer.concat([hmac, nonce]).toString('hex'); // 80个字符
```

**错误示例**：
```javascript
// ❌ 错误：使用URL编码后的路径
const path = '/api/v1/files/hxljzz/images/20250710/%E5%BE%AE%E4%BF%A1%E5%9B%BE%E7%89%87_example.jpg';
```

### 两种调用方式

#### 方式1：路径中包含存储类型（推荐）
```bash

curl -X DELETE "http://localhost:8080/api/v1/files/hxljzz/images/20250710/example.jpg?expires=1735977600&signature=abc123..."

# 删除 s3 存储中的文件
curl -X DELETE "http://localhost:8080/api/v1/files/s3/documents/2024/report.pdf?expires=1735977600&signature=abc123..."

# 删除默认存储中的文件
curl -X DELETE "http://localhost:8080/api/v1/files/images/20250710/example.jpg?expires=1735977600&signature=abc123..."
```

```bash
# 通过 storage 参数指定存储类型

```

**响应示例**：
```json

  "success": true,
  "message": "文件删除成功"
}
```

## 🛠️ 客户端SDK示例

### Go 客户端

```go
package main

import (
    "bytes"
    "crypto/hmac"
    "crypto/sha256"
    "encoding/hex"
    "fmt"
    "io"
    "mime/multipart"
    "net/http"
    "os"
    "time"
)

type FileUploaderClient struct {
    BaseURL   string
    SecretKey string
}

func (c *FileUploaderClient) generateSignature(method, path string, expires int64) string {
    message := fmt.Sprintf("%s|%s|%d", method, path, expires)
    h := hmac.New(sha256.New, []byte(c.SecretKey))
    h.Write([]byte(message))
    return hex.EncodeToString(h.Sum(nil))
}

func (c *FileUploaderClient) UploadFile(filePath, storage string) error {
    file, err := os.Open(filePath)
    if err != nil {
        return err
    }
    defer file.Close()

    var body bytes.Buffer
    writer := multipart.NewWriter(&body)

    part, err := writer.CreateFormFile("file", filePath)
    if err != nil {
        return err
    }

    _, err = io.Copy(part, file)
    if err != nil {
        return err
    }

    if storage != "" {
        writer.WriteField("storage", storage)
    }

    writer.Close()

    expires := time.Now().Add(time.Hour).Unix()
    signature := c.generateSignature("POST", "/api/v1/upload", expires)

    url := fmt.Sprintf("%s/api/v1/upload?expires=%d&signature=%s",
        c.BaseURL, expires, signature)

    req, err := http.NewRequest("POST", url, &body)
    if err != nil {
        return err
    }

    req.Header.Set("Content-Type", writer.FormDataContentType())

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        return err
    }
    defer resp.Body.Close()

    fmt.Printf("上传响应状态: %s\n", resp.Status)
    return nil
}

func main() {
    client := &FileUploaderClient{
        BaseURL:   "http://localhost:8080",
        SecretKey: "your-secret-key",
    }

    err := client.UploadFile("example.jpg", "protected_images")
    if err != nil {
        fmt.Printf("上传失败: %v\n", err)
    } else {
        fmt.Println("上传成功!")
    }
}
```

### Python 客户端

```python
import hashlib
import hmac
import time
import requests

class FileUploaderClient:
    def __init__(self, base_url, secret_key):
        self.base_url = base_url
        self.secret_key = secret_key

    def generate_signature(self, method, path, expires):
        message = f"{method}|{path}|{expires}"
        signature = hmac.new(
            self.secret_key.encode(),
            message.encode(),
            hashlib.sha256
        ).hexdigest()
        return signature

    def upload_file(self, file_path, storage=None):
        expires = int(time.time()) + 3600  # 1小时后过期
        signature = self.generate_signature("POST", "/api/v1/upload", expires)

        url = f"{self.base_url}/api/v1/upload"
        params = {
            "expires": expires,
            "signature": signature
        }

        files = {"file": open(file_path, "rb")}
        data = {}

        if storage:
            data["storage"] = storage

        response = requests.post(url, params=params, files=files, data=data)
        files["file"].close()

        return response.json()

# 使用示例
client = FileUploaderClient("http://localhost:8080", "your-secret-key")
result = client.upload_file("example.jpg", "protected_images")
print(result)
```

### JavaScript/Node.js 客户端

```javascript
const crypto = require('crypto');
const FormData = require('form-data');
const fs = require('fs');
const axios = require('axios');

class FileUploaderClient {
    constructor(baseUrl, secretKey) {
        this.baseUrl = baseUrl;
        this.secretKey = secretKey;
    }

    generateSignature(method, path, expires) {
        const message = `${method}|${path}|${expires}`;
        const signature = crypto
            .createHmac('sha256', this.secretKey)
            .update(message)
            .digest('hex');
        return signature;
    }

    async uploadFile(filePath, storage = null) {
        const expires = Math.floor(Date.now() / 1000) + 3600; // 1小时后过期
        const signature = this.generateSignature('POST', '/api/v1/upload', expires);

        const formData = new FormData();
        formData.append('file', fs.createReadStream(filePath));

        if (storage) {
            formData.append('storage', storage);
        }

        const url = `${this.baseUrl}/api/v1/upload?expires=${expires}&signature=${signature}`;

        try {
            const response = await axios.post(url, formData, {
                headers: formData.getHeaders()
            });
            return response.data;
        } catch (error) {
            throw new Error(`上传失败: ${error.message}`);
        }
    }
}

client.uploadFile('example.jpg', 'protected_images')
    .then(result => console.log('上传成功:', result))
    .catch(error => console.error('上传失败:', error));
```

## 🖼️ 图片优化功能

File Uploader 支持动态图片优化，可以在访问图片时通过URL参数进行实时处理。

### 支持的优化参数

- `w` - 宽度（像素）
- `h` - 高度（像素）
- `q` - 质量（1-100）
- `f` - 输出格式（jpeg、png、webp、avif）

### 使用示例

```bash
# 原始图片
http://localhost:8080/uploads/image.jpg

# 调整尺寸
http://localhost:8080/uploads/image.jpg?w=800&h=600

# 调整质量
http://localhost:8080/uploads/image.jpg?q=80

# 转换格式为AVIF
http://localhost:8080/uploads/image.jpg?f=avif

# 组合使用：调整尺寸、质量并转换为AVIF格式
http://localhost:8080/uploads/image.jpg?w=800&h=600&q=80&f=avif
```

### 支持的输出格式

- **JPEG** (`f=jpeg` 或 `f=jpg`) - 适合照片，文件较小
- **PNG** (`f=png`) - 支持透明度，适合图标和图形
- **WebP** (`f=webp`) - 现代格式，压缩率高
- **AVIF** (`f=avif`) - 最新格式，压缩率最高，支持HDR

### 配置文件类型控制

在 `config/config.yaml` 中可以灵活配置允许的文件类型：

```yaml
upload:
  max_file_size: 104857600    # 100MB
  allowed_extensions:
    # 通配符模式
    - "*"                     # 允许所有文件类型
    - "image/*"               # 允许所有图片类型
    - "video/*"               # 允许所有视频类型
    - "application/pdf"       # 允许特定MIME类型

    # 具体扩展名
    - ".jpg"
    - ".png"
    - ".avif"                 # 支持AVIF格式
    - ".pdf"
```

**通配符说明**：
- `*` - 允许所有文件类型
- `image/*` - 基于MIME类型匹配所有图片
- `video/*` - 基于MIME类型匹配所有视频
- `application/pdf` - 匹配特定MIME类型
- `.jpg` - 匹配特定扩展名

## ⚠️ 注意事项

### 1. 签名安全
- 密钥必须保密，不要在客户端代码中硬编码
- 签名有过期时间，建议设置合理的过期时间
- 定期更换密钥以提高安全性

### 2. 文件大小限制
- 默认最大文件大小为 100MB
- 可在配置文件中调整限制
- 大文件上传建议使用分片上传

### 3. 存储配置
- 确保存储配置正确且可访问
- S3 存储需要正确的访问密钥和权限
- 本地存储需要确保目录权限正确

### 4. 错误处理
- 所有 API 调用都应该包含错误处理
- 检查响应状态码和错误信息
- 实现重试机制处理临时性错误

### 5. 性能优化
- 使用缩略图减少带宽消耗
- 实现客户端缓存机制
- 批量操作比单个操作更高效
